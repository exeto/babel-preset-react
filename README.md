# babel-preset-react

> Babel preset for react app

**THIS PACKAGE IS NO LONGER MAINTAINED**

## Install

```sh
$ yarn add core-js@^2.5.7
$ yarn add --dev @exeto/babel-preset-react
```

## Usage

Add following in your `.babelrc`:

```json
{
  "presets": ["@exeto/babel-preset-react"]
}
```
